semantique.geometries¶

semantique.geometries(value, **kwargs)[source]¶

Spatial geometries.

Special value representing one or more spatial vector geometries, such as points, lines or polygons. Can be used to apply spatial filters to data cubes.

Parameters

  • value – One or more spatial features containing the geometries. Should be given as an object that can be understood by the initializer of SpatialExtent. This includes geopandas.GeoDataFrame objects.

  • **kwargs – Additional keyword arguments passed on to the initializer of SpatialExtent.

Returns

dict – JSON-serializable object that contains the geometries, and can be understood by the query processor as such.
